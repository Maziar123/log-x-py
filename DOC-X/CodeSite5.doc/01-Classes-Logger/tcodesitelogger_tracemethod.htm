Remarks

Use TraceMethod to immediately send an EnterMethod message, and when control exits from the current method whether by normal control flow or by exception, an ExitMethod message is automatically sent.

The primary version of this method simply takes the name of the method as a parameter. The second version takes an object parameter and will automatically extract the class name of the object and the value of its "Name" property if one exists and displays this additional information along with the method name.

The Option parameter controls whether or not timing information will be included in the method trace. The Option parameter can be set to either tmoTraceOnly or tmoTiming. If tmoTiming is specified, then the format of the timing message is affected by the TimingFormat parameter.  TimingFormat can be one of the following values: tfStopwatch, tfClockCycles, tfMilliseconds, or tfMicroseconds.

Example

procedure TSample.SampleMethod;

begin

  CodeSite.TraceMethod( Self, 'SampleMethod' );   // EnterMethod message sent

 

  // Method implementation details

 

end;  // ExitMethod message sent when method exits
