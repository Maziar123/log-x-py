TCodeSiteLogger.SendMemoryAsHex Method

    

Sends the contents of a memory buffer as a CodeSite Memory message.

Remarks

Use this method to send the contents of the specified Buffer parameter to the current destination.  The bytes referenced by the Buffer pointer are formatted in a typical hex-view format.  Unlike the SendFileAsHex and SendStreamAsHex methods, you must specify the size of the buffer in the Size parameter.  This method is useful for inspecting the bytes of a memory block.  When the resulting message is viewed in a CodeSite viewer, the Inspector Pane shows the hex-view contents of the buffer.  For example, the following is a typical 8 byte-value column hex view.
