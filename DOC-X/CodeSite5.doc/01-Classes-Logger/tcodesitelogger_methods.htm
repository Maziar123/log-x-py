Logging Classes

T|CodeSiteLogger Methods

    

Methods

Name

Description

AddCheckPoint

Increments the check point counter and sends the value as a CodeSite CheckPoint message.

AddSeparator

Sends a CodeSite Separator message to visually separate messages in the resulting log.

Assert

Evaluates the specified expression and sends the string message as a CodeSite Error message if the expression fails.

Clear

Instructs the CodeSite Viewer to clear all its messages.

CloseDispatcher

Closes the local CodeSite Dispatcher if it is running.

ConnectUsingCopyData

Override the CodeSiteManager's default connection by connecting to the local CodeSite Dispatcher using wm_CopyData.

ConnectUsingTcp

Override the CodeSiteManager's default connection by connecting to a local or remote CodeSite Dispatcher using TCP.

EnterMethod

Sends the name of a method in a CodeSite EnterMethod message to indicate the start of a method block.

ExceptionHandler

Built-in exception handler that automatically sends unhandled exceptions.

ExitMethod

Sends the name of a method as CodeSite ExitMethod message to indicate the end of a method block.

ExitMethodCollapse

Sends the name of a method as CodeSite ExitMethod message to indicate the end of a method block. The CodeSite Viewer will then collapse the completed method block.

Installed

Determines if CodeSite Tools are installed.

LogError

Writes the specified message string as an Error entry in the Event Log.

LogEvent

Writes the specified message string as an Information entry in the Event Log.

LogWarning

Writes the specified message string as a Warning entry in the Event Log.

ResetCheckPoint

Resets the check point counter to 0.

Send

Sends a message string along with some data value.

SendAssigned

Sends the assignment state of the specified object.

SendColor

Sends the specified TColor as a CodeSite Color message.

SendCollection

Evaluates the specified IEnumerable collection and sends its contents as a CodeSite Collection message.

SendComponentAsText

Sends the text stream representation of the specified component as a CodeSite Component As Text message.

SendComponents

Sends the Components hierarchy for the specified component as a CodeSite Custom Hierarchy message.

SendControls

Sends the Controls hierarchy for the specified control as a CodeSite Custom Hierarchy message.

SendCurrency

Sends a Currency value.

SendCustomData

The primary version of this method utilizes the properties and methods of ICodeSiteCustomData to obtain custom formatted data from the specified object.

To handle non-object-based types in Delphi/C++Builder, an alternate version is provided to invoke the TCodeSiteFormatter class associated with the specified message type to format the untyped data parameter and sends the formatted result.

SendDateTime

Sends a date-time value.

SendDateTimeIf

Sends a date-time value if the specified expression is True.

SendEnum

Sends the symbol name corresponding to the specified enumerated value.

SendError

Sends a CodeSite Error message.

SendException

Records the details of the specified Exception object and sends the data as a CodeSite Exception message.

SendFileAsHex

Sends the contents of the specified file as CodeSite Memory message.

SendFileVersionInfo

Sends the specified file's VersionInfo block as a CodeSite Version Information message.

SendFmtMsg

Sends a formatted CodeSite message.

SendHeapStatus

Sends the current heap statistics as a CodeSite Memory Status message.

SendIf

Sends a CodeSite message only if the specified expression is true.

SendKey

Sends information about the specified key code.

SendMemoryAsHex

Sends the contents of a memory buffer as a CodeSite Memory message.

SendMemoryStatus

Sends the system's current memory status as a CodeSite Memory Status message.

SendMouseButton

Sends the details of the current mouse button event.

SendMsg

Sends a simple CodeSite message.

SendMsgDetails

General purpose method for sending a CodeSite message with additional details.

SendNote

Sends a CodeSite Note message.

SendParents

Sends the parent hierarchy for the specified control as a CodeSite Custom Hierarchy message.

SendPointer

Sends the memory address of the specified pointer.

SendProperty

Scans the specified object for the specified property name and if present sends the property's value as a CodeSite message.

SendRegistry

Sends the contents of the specified Registry key as a CodeSite Registry message.

SendReminder

Sends a CodeSite Reminder message.

SendScreenShot

Sends an image of the specified window as a CodeSite Bitmap message.

SendSet

Sends the elements of a set.

SendStackTrace

Sends a stack trace based on the current execution point as a CodeSite Stack Trace message.

SendStreamAsHex

Sends the contents (in hex format) of the specified stream as a CodeSite Memory message.

SendStreamAsText

Sends the contents (in text format) of the specified stream as a CodeSite Memory message.

SendSystemInfo

Sends information about the user's system in a CodeSite System Information message.

SendTextFile

Sends the contents of the specified text file as a CodeSite Text File message.

SendVariant

Sends the value of the specified variant (as a string).

SendVersionInfo

Sends the application's VersionInfo block as a CodeSite Version Information message.

SendWarning

Sends a CodeSite Warning message.

SendWindowHandle

Sends low-level window-related information (i.e. class, styles, etc.) for the specified window handle as a CodeSite Window Handle message.

SendWinError

Converts a Windows Error Code into a formatted message string and then sends the string as a CodeSite Error message.

SendXmlData

Sends the contents of the specified string as a CodeSite XML message.

SendXmlFile

Sends the contents of the specified XML file as a CodeSite XML message.

Timer

Creates a profiling timer that can be used to measure the time it takes to execute a section of code.

TraceMethod

Immediately sends an EnterMethod message, and when control exits from the current method whether by normal control flow or by exception, an ExitMethod message is sent automatically.

TraceMethodCollapse

Same as TraceMethod except the CodeSite Viewer will then collapse the completed method block.

UseCodeSiteManagerConnection

Removes any existing connection override and once again uses the connection that is defined by the CodeSiteManager.

Write

Writes the values of the specified parameters to the appropriate row in the Viewer's ScratchPad.

WriteColor

Writes the RGB values of the specified TColor parameter to the appropriate row in the Viewer's ScratchPad.

WriteCurrency

Writes the value of the specified Currency parameter to the appropriate row in the Viewer's ScratchPad.

WriteDateTime

Writes the value of the specified TDateTime parameter to the appropriate row in the Viewer's ScratchPad.
