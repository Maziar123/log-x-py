TCodeSiteLogger.Timer Method

    

Creates a profiling timer that can be used to measure the time it takes to execute a section of code.

Frameworks

Delphi

.NET

Syntax

Delphi

function Timer( const Description: string; TimingFormat: TCodeSiteTimingFormat = tfMicroseconds;

                AutoStart: Boolean = True; AutoStop: Boolean = True ): ICodeSiteTimer;

C++Builder

_di_ICodeSiteTimer __fastcall Timer( const System::UnicodeString Description, 

                                     TCodeSiteTimingFormat TimingFormat = (TCodeSiteTimingFormat)(0x3), 

                                     bool AutoStart = true, bool AutoStop = true)/* overload */;

Remarks

Use this method to create a new profiling timer. Specify a brief description of what you are timing in the first parameter. The TimingFormat parameter controls the format of the resulting time duration. Possible values are: tfStopwatch, tfClockCycles, tfMilliseconds, and tfMicroseconds.

When AutoStart is True, the timer starts when it is called. If AutoStart is False, then the timer will begin when the Start method is called. Likewise, when AutoStop is True, the timer stops when the current method exits. If AutoStop is False, then the timer will end when the Stop method is called. When the timer ends, the resulting time duration is automatically sent as a csmTiming message.

Example

var

  Timer: ICodeSiteTimer;

begin

  Timer := CodeSite.Timer( 'Loop Timer', tfMilliseconds, False, False );

 

  Timer.Start;

  for I := 1 to 10000 do

    // Do something in loop;

  Timer.Stop;                 // Timing message sent

end;
