CodeSite Dispatcher

Status

    

The CodeSite Dispatcher displays different icons in the system tray notification area to show message activity and other status information.  The following is a list of the possible icons that may be displayed by the CodeSite Dispatcher.

Dispatcher is running normally and may be routing a few messages.

Dispatcher is active routing messages to destinations.

Dispatcher is handling a moderate workload of messages.

Dispatcher is handling a heavy workload of messages.

The Dispatcher is currently disabled and is ignoring all incoming messages.

An error message has been added to the Dispatcher Log.  Select the View Dispatcher Log menu item to view the log.
