CodeSite Live/File Viewer

Layout: Inspector Pane

    

To display the Inspector, use the Inspector command.

The Inspector pane is used to display any data that may be attached to a CodeSite message.  For example, when an Object message is selected in the Message Log, the Inspector pane displays a name-value grid of all the object's properties and their current values at the time the message was sent.

Custom Inspectors

In addition, the inspector pane is where custom inspectors are displayed.  Custom Inspectors are added to the CodeSite Viewer through a plug-in architecture.  CodeSite comes with several Custom Inspectors including:

Compare Objects Inspector

This inspector is invoked when two Object messages are selected in the message log and shows the differences in the published property values that exist between the two objects.

Compare Custom Properties Inspector

This inspector is invoked when two CustomProperties messages are selected in the message log and shows the differences in published property values that exist between two CustomProperties messages.

Compare Memory Status Inspector

This inspector is invoked when two MemoryStatus messages are selected in the message log and shows the differences between the memory status snapshots.
