CodeSite Live/File Viewer

Commands: Tools|Options - General

    

Show Clear Views Confirmation

Check this option to instruct the Viewer to display a confirmation dialog box whenever the message list is cleared.

Show Clear Bookmarks Confirmation

Check this option to instruct the Viewer to display a confirmation dialog box whenever the current set of bookmarks are cleared.

Auto-Save Project Changes on Exit

Check this option to instruct the Viewer to automatically save any changes that you make to the currently loaded project when you exit the CodeSite Viewer.  Turning this option off is useful in situations where you want to try out different filtered views but do not want to modify your project file settings.

Auto-Collapse Empty Methods

Check this option to automatically collapse empty EnterMethod and ExitMethod message pairs that are received by the Viewer.

Flatten Structure

Check this option to remove all hierarchical structure from the message log.

Auto-Scroll New Messages (Live Viewer Only)

Check this option to instruct the Live Viewer to scroll the list as new messages arrive so that the new message is in view.

Highlight Color and Highlight Font Color

Use these two color pickers to select the background and font color to be used to highlight selected text throughout the viewer.

Value Color

Use the color picker to select the font color to be used for display values in the Message Detail Panel and stock inspectors such as the Object Inspector.

Bookmark Color

Use the color picker to select the background color to be used for highlighting marked lines in the message list.

Sync Msg Color

Use the color picker to select the background color to be used for highlighting the message to be synchronized while Synchronize Message Mode is in effect.

Shade Alternate Rows

Check this option to have alternate rows in the message log shaded by alpha blending the Blending Color value with the background color of the message. The shading also applies to messages that specify a category color.
