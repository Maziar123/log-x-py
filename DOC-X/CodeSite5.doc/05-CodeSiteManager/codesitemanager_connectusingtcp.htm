Remarks

Use the CodeSiteManager.ConnectUsingTcp method to have all T|CodeSiteLogger instances connect to a local or remote CodeSite Dispatcher using the TCP protocol. Using the ConnectUsingTcp method is required when generating CodeSite messages from services and ASP.NET.  Please read Logging from Services for more information.

Delphi & C++Builder

The MsgFormat parameter controls the format of the message as it is sent to the Dispatcher.  The default value is mfDefault. Pass in mfCompressed and messages are compressed before they are transported to the Dispatcher. Use the mfCompressed message format along with the new Compressed options for destinations and the message data will be compressed and obfuscated throughout the entire transport process.

Example

// Connect to local Dispatcher using default port

CodeSiteManager.ConnectUsingTcp; 

 

// Connect to the HelpDesk machine on port 4040.

// The Dispatcher must be configured to monitor port 4040 for TCP messages.

CodeSiteManager.ConnectUsingTcp( 'HelpDesk', 4040 );
