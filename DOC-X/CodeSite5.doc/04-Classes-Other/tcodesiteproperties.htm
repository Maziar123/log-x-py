Remarks

The nonvisual TCodeSiteProperties component lets developers modify the various properties of the global TCodeSiteLogger instance CodeSite at design-time. Simply drop a TCodeSiteProperties component on your main form or data module and set the appropriate properties.  When the form or data module is loaded, the properties stored in the TCodeSiteProperties component are transferred to the global TCodeSiteLogger instance, CodeSite.

The major advantage of the TCodeSiteProperties component is the ability to modify the Destination property at design-time.  Having all of the properties available in the Object Inspector makes it easier to customize any of the available properties.  In addition, the TCodeSiteProperties component makes it easy to write an event handler for the OnSendMsg event.

Please note that the TCodeSiteProperties component only provides access to the properties and events of the global TCodeSiteLogger instance, CodeSite--it does not provide access to the methods defined in the TCodeSiteLogger class. TCodeSiteProperties descends from TComponent and not TCodeSiteLogger.  To call the various methods, simply use the global TCodeSiteLogger instance, CodeSite.

IMPORTANT

Only one instance of this component should be created in an application.  If additional instances of this component are created, the global TCodeSiteLogger instance, CodeSite, will take on the values defined in the most recent TCodeSiteProperties instance.
