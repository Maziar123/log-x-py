CodeSite Destination String Builder

    

The Destination String Builder is a small utility program (located in CS5\Bin) for creating Destination Strings that can be used in setting the RemoteDestinationString property in T|CodeSiteDestination.TCP and T|CodeSiteDestination.UDP.

NOTE

Delphi/C++Builder developers can initialize the sub-properties of a TCodeSiteDestination instance by using the AsString property and setting it to a destination string.

Each tab in the main window of the Destination String Builder provides access to all the sub-properties of the T|CodeSiteDestination class: Viewer, LogFile, TCP, UDP, HTTP, and Alias.  To activate a particular destination type, simply check the box at the top of the destination type page. Then use the controls provide to specify the various property values.  The Destination String area at the bottom of the window displays the current string.  If there are any missing values, the destination string will contain ?? as place-markers.

Please see the Destination Strings topic for more details.
