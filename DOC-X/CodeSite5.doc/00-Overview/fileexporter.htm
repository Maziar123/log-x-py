CodeSite File Exporter

    

The CodeSite File Exporter (CSFileExporter.exe) is a command line utility (located in CS5\Bin) for converting native CodeSite log files into other formats.  Specifically, XML (the default), tabbed delimited text, or comma separated text.  Executing the CSFileExporter at the command line displays the following:

Usage: CSFileExporter [options] filename.csl

Options:

 -x = Export to XML file (Default)

 -t = Export to Tabbed Delimited Text file

 -c = Export to Comma Separated Values file
