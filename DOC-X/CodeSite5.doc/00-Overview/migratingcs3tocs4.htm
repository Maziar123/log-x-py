Migrating from Older Versions

Migrating from Version 3.x

    

T|CodeSiteObject replaced with T|CodeSiteLogger

The primary logging class used to capture, record, and transmit CodeSite logging messages has been changed from T|CodeSiteObject to T|CodeSiteLogger. The old classes were deprecated in CodeSite 4 and have been removed in CodeSite 5.  This change does NOT affect the global CodeSite object in Delphi/C++Builder, nor the static class in .NET.

TransportMethod and Dispatcher Connections

The TransportMethod property has been removed from T|CodeSiteLogger. Configuring the connection to the CodeSite Dispatcher has been replaced with several new methods defined in the CodeSiteManager and in the T|CodeSiteLogger class.  Please see the Dispatcher Connections topic for more information.

To change the connection mechanism used by ALL T|CodeSiteLogger instances, use one of the following CodeSiteManager method:

ConnectUsingCopyData;

ConnectUsingTcp( host, port );

ConnectUsingTcp( host );

ConnectUsingTcp;
 

A T|CodeSiteLogger instance may override the default connection mechanism by using one of the following methods:

ConnectUsingCopyData;

ConnectUsingTcp( host, port );

ConnectUsingTcp( host );

ConnectUsingTcp;

 

NOTE

It should also be noted that the ConnectUsingTcp methods allow a direct connection to be established to a remote CodeSite Dispatcher.

Package File Changes

The packages that contain the CodeSite Logging classes for Delphi/C++Builder development have been renamed according to the Raize Software package naming convention.  The primary benefit of this is that the version number of the product and the version number of the RTL have been removed from the names.  The result is that when a new version of CodeSite is released, you will no longer be required to modify the source code of your dependent packages.

The new packages are:

CodeSiteLoggingPkg120.bpl

CodeSiteLoggingPkg140.bpl

CodeSiteLoggingPkg150.bpl

CodeSiteLoggingPkg160.bpl

However, in your dependent packages, you simply need to list CodeSiteLoggingPkg.dcp as the required package.
