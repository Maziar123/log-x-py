Advanced Techniques - CodeSite and .NET

Trace Listener

    

CodeSite is similar in concept to the System.Diagnostics.Trace class built into the Microsoft .NET Framework. However, CodeSite is much more powerful and flexible.  If you are currently using Trace in your applications, you can quickly and easily have your Trace output go to the same destination as the CodeSite static class (by default the CodeSite Live Viewer). Simply add the following statement at the beginning of your application code.

CodeSiteManager.TraceListenerEnabled = true;

 

Please note that although this approach provides a quick way to get started using CodeSite, we recommend that you use the CodeSite class instead of Trace to get the full benefits of the CodeSite Logging System.
