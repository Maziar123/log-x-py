Technical Support

    

In addition to this electronic documentation, Raize Software offers free technical support via internet web forums and email. When requesting technical support, please be sure to include a detailed description of the problem.  Also, please provide the following information:

·Version/Build Number of CodeSite (e.g. 5.1.2)

·Operating System and Version Number

·Version number of development tool (i.e. Embarcadero RAD Studio 10.1 Berlin, Visual Studio 2012, Delphi XE2, etc.)

IMPORTANT

Technical support will only be provided to registered users.  When sending technical support questions, please include your name and company name (if applicable).
