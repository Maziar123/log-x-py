T|AliasDestination Class

    

This class is used to manage the details that define an Alias destination.

Remarks

This destination type instructs the CodeSite Dispatcher to obtain the destination information to use from the CSDispatcher.ini file.  Alias names correspond to sections in the INI file and each section defines a DestinationDetails entry, which is a string representation of a Destination. Please read Destination Strings for more information.

NOTE

The Alias details are only utilized when the Alias.Active property is set to True.

Example

Consider the following section in the CSDispatcher.ini file:

[Sample]

DestinationDetails=Viewer,File[Path=C:\Logs\Sample.csl;MaxSize=256;MaxParts=4]

 

and the following Destination specification:

Alias.Name := 'Sample';

 

DestinationDetails represents the string representation of the all the destination properties of a T|CodeSiteDestination instance.  Please read the topic on Destination Strings for more information.
